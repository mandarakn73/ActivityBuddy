⚠️ EmailJS Setup Required (one-time, free)

  1. Sign up at emailjs.com → free, no credit card
  2. Add Email Service → Gmail → Connect (OAuth, no password given)
  3. Create Email Template — set fields exactly as below:
To Email:{{to_email}} ← REQUIRED
Subject:{{subject}}
Body:{{html_body}} — set Content Type to HTML, Auto Escape OFF
  1. Paste your Public Key, Service ID, Template ID in ⚙️ Settings → EmailJS Config
